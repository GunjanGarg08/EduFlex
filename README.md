EduFlex is a web-based Learning Management System (LMS) designed to provide a centralized platform for online course delivery. The system allows learners to browse, purchase, and access video-based courses, while educators can create, upload, and manage their course content along with pricing and enrollments. The goal of EduFlex is to simplify online learning by providing secure access, structured course management, and seamless payment integration.

The platform is developed using the MERN stack (MongoDB, Express.js, React.js, Node.js) for a scalable, performance-driven architecture. Clerk is used for secure user authentication and role-based access, ensuring smooth login and account handling for both learners and educators. Stripe is integrated for safe course payment processing, enabling automatic enrollment after successful transactions. Course videos and media content are stored and delivered through Cloudinary, ensuring reliable performance and optimized streaming.

EduFlex focuses on accessibility, user-friendly interaction, and efficient course handling. It offers learners a simple way to access purchased courses and educators a streamlined process to publish content and track enrollments. With modular development, secure integrations, and scalable architecture, EduFlex provides a structured and practical solution for modern online education needs.
